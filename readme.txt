В рамках дз7 сформирована документация с использованием Sphinx для модулей server и client (см. папку documentation).
Потратил очень много времени, чтобы все заработало (из-за мелких ошибок, которые Sphinx объяснять не умеет). 
В результате, Sphinx показался мне неоправданно сложным и сыроватым, с учетом того, что он призван облегчать труд разработчика.
